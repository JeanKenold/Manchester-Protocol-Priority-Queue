# Simulador de Atendimento Hospitalar (Protocolo de Manchester) 🏥

Este projeto é um simulador de triagem hospitalar desenvolvido para a disciplina de **Estrutura de Dados**. O sistema utiliza **Filas com Prioridade** para organizar o atendimento de pacientes com base no risco clínico.

## 📋 Funcionalidades

- **Cadastro de Pacientes:** Nome, idade e sintomas.
- **Triagem:** Classificação baseada em 5 níveis de prioridade (Vermelho a Azul).
- **Atendimento Otimizado:** O sistema prioriza sempre o caso mais crítico.
- **Relatórios:** Estatísticas de tempo de espera e total de atendimentos por risco.

## 🛠️ Tecnologias Utilizadas

- Linguagem C
- Estruturas de Dados: Filas Dinâmicas, Filas de Prioridade.
